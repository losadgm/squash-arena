# Squash Arena

A 3D pong game with a vaporwave aesthetic built with React, TypeScript, and Three.js.

## How to play

Move your mouse to control the cyan paddle. Keep the ball from getting past you while trying to score against the opponent. First to 7 points wins.

## Tech

- **Three.js** — 3D rendering with bloom and SMAA post-processing
- **React + Zustand** — UI and state management
- **Web Audio API** — procedural sound effects, no audio files

## Run locally

```bash
npm install
npm run dev
```
